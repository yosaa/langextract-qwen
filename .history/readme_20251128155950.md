# 文档纠错功能使用说明

## 功能概述

[示例图片](/img/2.png)

新增的文档纠错功能可以自动检测文档中的三类错误：
- **错别字**：拼写错误或同音字误用
- **语病**：语法不通顺、表达不清晰的地方
- **标点符号错误**：标点符号使用不当或缺失

检测结果会以交互式HTML页面展示，点击右侧错误列表可高亮显示原文中的错误位置，并显示建议的修改方案。

## 使用方法

### 1. 基本用法

```bash
python error_check.py 文件名.docx
```

### 2. 示例

```bash
python error_check.py 合同文档.docx
```

### 3. 输出文件

程序运行后会生成两个文件：
- `error_check_result.jsonl` - JSON Lines格式的结构化结果
- `error_check_visualization.html` - 交互式可视化页面（双击打开）

# 合同信息抽取功能使用说明

## 功能概述
[示例图片](/img/1.png)

- 从合同文本中提取关键信息（如合同编号、甲方名称、乙方名称、身份证号、联系电话、金额等），并按在原文中的顺序展示。
- 结果以交互式HTML页面展示，点击右侧抽取项可高亮原文位置并联动定位。

## 文件说明

- `main.py` - 合同信息抽取主程序
- `template.html` - 可视化页面模板
- `contract_visualization.html` - 生成的可视化结果页面（运行后生成）
- `contract_extraction.jsonl` - 结构化抽取结果（运行后生成）

## 使用方法

```bash
python main.py 文件名.docx
```

## 可视化页面功能

- 左侧面板：原始合同文本，高亮定位到已抽取的字段位置。
- 右侧面板：抽取结果列表与数量统计，支持点击联动高亮与平滑滚动定位。
